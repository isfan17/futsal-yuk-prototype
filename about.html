Futsal Yuk!

Aplikasi pencari teman futsal secara online. Dari Futsal Yuk! untuk para penggemar futsal Indonesia

Futsal Yuk! adalah sebuah aplikasi yang cocok digunakan oleh para penggemar futsal untuk mencari orang untuk diajak bermain futsal. Alasan kami membuat aplikasi ini adalah karena banyaknya para penggemar futsal yang sudah mulai sulit mencari teman untuk diajak bermain futsal. Oleh karena itu, kami hadir untuk mempermudah para penggemar futsal mencari orang untuk diajak bermain futsal bersama dimanapun mereka berada.
